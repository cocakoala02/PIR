# PIR 医疗隐私查询演示（课程版）

一个最小可运行的 Web 演示：
- 客户端输入（选择）罕见病代码。
- 客户端本地生成 Paillier 密钥并加密 one-hot 索引。
- 服务端仅收到加密索引并进行同态聚合，无法直接识别病种。
- 客户端解密后得到治疗方案。

## 运行方式

```bash
node server.js
```

浏览器访问：

```text
http://localhost:8090
```

## 演示要点

1. 点击“生成本地密钥”。
2. 选择疾病代码，点击“执行 PIR 查询”。
3. 观察页面右侧日志，出现：
   - `收到一组加密索引（长度 X），无法识别具体疾病名称。`

## 说明

- 本项目是教学用途的“简化 PIR”实现，参数规模较小，不可用于真实医疗生产环境。
- 目标是展示密码学作业中的核心思想：查询索引隐匿、服务端不可见。
