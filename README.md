# PIR 医疗隐私查询系统

一个可运行的 Web 页面：
- 客户端输入（选择）罕见病代码。
- 客户端本地生成 Paillier 密钥并加密 one-hot 索引。
- 服务端仅收到加密索引并进行同态聚合，无法直接识别病种。
- 客户端解密后得到治疗方案。

## 运行方式

```bash
node server.js
```

浏览器访问：

```text
http://localhost:8090
```

## 使用步骤

1. 点击“生成本地密钥”。
2. 选择疾病代码，点击“执行 PIR 查询”。
3. 观察页面右侧日志，出现：
   - `收到一组加密索引（长度 X），无法识别具体疾病名称。`
